## 🚀 Quick Start (5 steps)

### Step 1: Open Terminal
```
c:\Users\srishti\My project (1)\ecommerce-store
```

### Step 2: Install Dependencies
```bash
npm install
```

### Step 3: Load Sample Data
```bash
npm run seed
```

**Creates:**
- Admin account: `admin@ecommerce.com` / `admin123`
- 12 sample products
- Ready-to-use database

### Step 4: Start Backend
```bash
npm run dev
```

**Server runs at:** `http://localhost:5000`

### Step 5: Open Frontend
- Open `client/index.html` with Live Server
- Or: `python -m http.server 3000` from `client/` folder
- Browser opens at: `http://localhost:3000`

**✅ Done!** You now have a fully functional e-commerce store running locally!

## 🚀 Deployment Quick Steps

### Deploy Backend to Render

1. Push to GitHub
2. Create Render account
3. New Web Service
4. Connect GitHub repo
5. Configure:
   - Build: `npm install`
   - Start: `npm start`
6. Add environment variables
7. Deploy

**Result:** `https://your-app.onrender.com/api/health`

### Deploy Frontend to Vercel

1. Create Vercel account
2. Import repository
3. Set root directory: `client`
4. Deploy
